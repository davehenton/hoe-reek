⁠4. Benutzung

hoe-reek integriert einen neuen Rake-Task:
  • rake reek
rake reek startet Reek mit den folgenden Parametern:
  • config_file: config.reek
  • source_files = 'lib/**/*.rb'
  • reek_opts = ''
  • fail_on_error = false
  • verbose = true
. Ausserdem erwartet reek eine Konfigurationsdatei config.reek"im Hauptverzeichnis des Projektes und überprüft alle Ruby Dateien in 'lib'. Keine weiteren Optionen werden an reek übergeben. Reek bricht bei einem Fehler nicht ab, und gibt eine vollständige Ausgabe.
Die Spezifikation der config.reek-Datei kann hier gefunden werden: https://github.com/troessner/reek#configuration-file
