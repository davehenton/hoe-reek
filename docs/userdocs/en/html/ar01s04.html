⁠4. Usage

hoe-reek integrates that new Rake-Task:
  • rake reek
rake reek starts reek with the following parameters:
  • config_file: config.reek
  • source_files = 'lib/**/*.rb'
  • reek_opts = ''
  • fail_on_error = false
  • verbose = true
So reek expects a config file config.reek in the main project directory and checks all Ruby files in 'lib'. No additional options will be given to reek. Reek doesn't break by a found error and gives out a verbose output.
The specification of config.reek can found there: https://github.com/troessner/reek#configuration-file
